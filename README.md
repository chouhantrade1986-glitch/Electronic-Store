# Electronic Store

A modern electronic store application built with React and cutting-edge web technologies.

## Features

- 🛍️ Browse and search electronic products
- 🛒 Shopping cart functionality
- 💳 Secure checkout process
- 👤 User authentication and profiles
- ⭐ Product ratings and reviews
- 📦 Order tracking
- 🔍 Advanced filtering and sorting

## Technologies Used

- **Frontend**: React, HTML, CSS, JavaScript (ES6+)
- **State Management**: Redux/Context API
- **Routing**: React Router
- **HTTP Client**: Axios/Fetch API
- **Backend**: Node.js/Express (if applicable)
- **Database**: MongoDB (if applicable)
- **Tools**: Git, GitHub, npm

## Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Steps

1. Clone the repository:
```bash
git clone https://github.com/chouhantrade1986-glitch/Electronic-Store.git
cd Electronic-Store
```

2. Switch to the react-development branch:
```bash
git checkout react-development
```

3. Install dependencies:
```bash
npm install
```

4. Create a `.env` file (if needed for API endpoints):
```bash
REACT_APP_API_URL=http://localhost:5000
```

5. Start the development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## Available Scripts

In the project directory, you can run:

### `npm start`
Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder.

### `npm run eject`
**Note: this is a one-way operation. Once you eject, you can't go back!**

## Usage

Navigate to `http://localhost:3000` to access the application. Browse products, add items to your cart, and proceed to checkout.
